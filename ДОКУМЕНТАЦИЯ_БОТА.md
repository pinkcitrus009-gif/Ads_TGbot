# D&D Master Bot v3 — Полная документация

## Что умеет бот

Телеграм-бот «Мастер подземелий» для игры в D&D 2024 на русском языке.  
Работает **в личных чатах** (одиночная игра) и **в групповых чатах** (многопользовательская партия).  
Персонаж создаётся один раз и переносится между чатами автоматически.

---

## Быстрый старт

### Личный чат (одиночная игра)
1. Напиши боту `/start` — пройди мастер создания персонажа
2. Когда персонаж готов — просто пиши, что делает твой герой
3. DM (ИИ) ответит и продвинет сюжет

### Групповой чат (партия)
1. Каждый игрок пишет боту в **личку** `/start` → создаёт персонажа
2. В групповом чате каждый пишет `/join` → вступает в партию
3. Бот автоматически перенесёт персонажа из личной переписки
4. В бою используй `/khod <действие>` для официальных ходов

## Все команды

### Персонаж

| Команда | Описание |
|---|---|
| `/start` | Создать персонажа (мастер: раса → класс → характеристики → имя) |
| `/info` | Показать лист персонажа. В группе — отправляет в личку |
| `/status` | Алиас `/info` |
| `/reset` | Сбросить персонажа и историю **в текущем чате**. Глобальные сохранения (`/saves`) сохраняются |
| `/summary` | Краткая летопись событий текущей сессии |

### Групповая игра

| Команда | Описание |
|---|---|
| `/join` | Вступить в партию. Если персонаж создан в личке — **автоматически копируется** |
| `/party` | Показать состав партии (имена, HP, уровни) |
| `/roll_initiative` | Бросить инициативу для всех → начать пошаговый бой |
| `/end_combat` | Завершить бой, сбросить очерёдность ходов |

---

### Система ходов в группе

| Команда | Описание |
|---|---|
| `/khod <действие>` | Официальное игровое действие **в свой ход** |
| `/turn <действие>` | Алиас `/khod` |
| `/pas` | Пропустить свой ход (защитная стойка) |
| `/skip` | Алиас `/pas` |
| `/done` | Передать ход вручную |

**Правила ходов:**
- Обычный текст в бою — свободный разговор (OOC, вне игры). DM не считает это ходом.
- `/khod` — это официальное действие. Если сейчас не твой ход — бот предупредит.
- Если DM задал уточняющий вопрос — ответь **реплаем на его сообщение**, потом ход продолжится.
- Если игрок не ходит 5 минут — ход автоматически пропускается.

### Боевые команды

| Команда | Описание |
|---|---|
| `/fight <противник>` | Начать бой с указанным врагом |

**Доступные навыки для `/check`:**
акробатика, атлетика, восприятие, выживание, история, запугивание, магия, медицина, обман, обращение с животными, природа, проницательность, религия, скрытность, убеждение, ловкость рук, расследование, выступление

### Квесты

| Команда | Описание |
|---|---|

---

### Сохранения (глобальные)

> Сохранения привязаны к **пользователю**, а не к чату.  
> Сохранил в личке — загружай в группе, и наоборот.  
> История и квесты группы при загрузке **не затрагиваются**.

| Команда | Описание |
|---|---|
| `/save <название>` | Сохранить персонажа под произвольным именем |
| `/load <название>` | Загрузить персонажа по имени (работает в любом чате) |
| `/saves` | Список всех твоих сохранений |

**Примеры:**
```
/save перед боссом
/save после прокачки
/load перед боссом
```

Кнопка **«💾 Сохранить»** в главном меню создаёт быстрое сохранение с именем `auto_DDMM_HHMM`.

---

### Отдых

| Действие | Эффект |
|---|---|
| **Короткий отдых** (кнопка 💤) | Бросок хит-дайса + мод. Телосложения → восстанавливает HP |
| **Долгий отдых** (кнопка 🌙) | Полное восстановление HP и все слоты заклинаний |

---

### Заклинания и слоты

- Слоты заклинаний отображаются в листе персонажа (`/info`)
- `/spell 1 Магическая стрела` — применяет заклинание 1-го уровня и тратит слот
- При долгом отдыхе все слоты восстанавливаются
- Классы с магией: Маг (полные слоты), Жрец, Бард, Паладин, Следопыт

## Перенос персонажа между чатами

### Автоматически при `/join`
Когда ты входишь в партию командой `/join`, бот проверяет, есть ли у тебя персонаж в другом чате.  
Если есть — копирует его автоматически и показывает уведомление:
> 🔄 Персонаж перенесён из другого чата!

### Через сохранения вручную
1. В личке: `/save мой_герой`
2. В группе: `/load мой_герой` — персонаж применится к текущей группе

### Что переносится, что остаётся
| Данные | При `/join` (авто) | При `/load` |
|---|---|---|
| HP, уровень, XP, золото | ✅ | ✅ |
| Инвентарь, способности, слоты | ✅ | ✅ |
| Характеристики (Сила, Ловкость…) | ✅ | ✅ |
| История группы | ❌ (локальная) | ❌ (не трогается) |
| Квесты группы | ❌ (локальные) | ❌ (не трогаются) |

## Как работает автоматическое обновление состояния

DM (ИИ) отслеживает события и автоматически обновляет:
- **HP** — урон и лечение
- **Золото** — нашёл клад, купил товар
- **XP** — опыт за победу или прохождение события
- **Предметы** — подобрал, потратил
- **Способности** — новые умения
- **Квесты** — добавляет новые, отмечает выполненные
- **Память DM** — важные факты о NPC, локациях, сюжете

После каждого важного события бот показывает итоговую строку изменений, например:
```
━━━━━━━━━━
🩸 -8 HP → 22/30
💰 +50 зм → 130 зм
⭐ +150 XP → 450/900
🎁 Получено: Ключ от тюрьмы
📋 Квест: Найти пленника [#3]
```

Уровни повышаются **автоматически** при накоплении XP:

| Уровень | XP |
|---|---|
| 1 → 2 | 300 |
| 2 → 3 | 900 |
| 3 → 4 | 2 700 |
| 4 → 5 | 6 500 |
| 5 → 6 | 14 000 |
| … | … |
| 19 → 20 | 355 000 |

При повышении уровня: прибавляется HP (хит-дайс + мод. Телосложения), обновляются слоты заклинаний.

## Типичные сценарии

### Новый игрок, впервые
```
/start → выбрать расу → выбрать класс → выбрать характеристики → ввести имя → подтвердить
```
После подтверждения DM начинает повествование автоматически.

### Перейти из личной игры в группу
```
(в группе)
/join
→ "🔄 Персонаж перенесён из другого чата!"
→ всё, можно играть
```

### Сохранить перед опасным боем
```
/save до_дракона
... (бой, может быть смерть) ...
/load до_дракона
```

### Начать заново, не теряя сохранения
```
/reset
→ "Персонаж сброшен. Сохранения сохранены."
/start → создать нового
```

## Советы

- **Пиши развёрнуто** — чем подробнее описано действие, тем интереснее ответ DM
- **Используй реплай** — когда DM задаёт уточняющий вопрос, отвечай реплаем на его сообщение
- **Сохраняйся перед боссами** — `/save` занимает секунду
- **В группе OOC-разговор** — обычный текст вне `/khod` не влияет на ход
- **Квесты добавляются автоматически** — если хочешь добавить вручную, используй `/quest add`
- **Инициатива** — всегда бросай `/roll_initiative` перед боем в группе для честного порядка ходов
